Enabling JavaScript in your web browser

Enabling JavaScript in your browser allows websites to make use of the powerful programming language behind much of the web.

JavaScript can be used to power various features, ranging from the deceptively simple press of an "Okay" button, all the way up to games and informational services.

Choose your browser below to get started.
  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

Google Chrome

Enabling JavaScript in Google Chrome can be accomplished by using the following steps:

  1. Open Chrome's settings menu through the menu at the top right of the navigation bar

  2. Type "java" into the search box and press enter

  3. Select "Site Settings"

  4. In the Site Settings menu, scroll down and select "JavaScript"

  5. Select "Sites can use JavaScript"

  6. JavaScript is now enabled on your browser!
